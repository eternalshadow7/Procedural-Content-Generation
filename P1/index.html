1. Press 'c' to clear the canvas.

  2. Hold the left DOWN KEY to generate a flipped picture (clear canvas first).

  3. Hold the UP ARROW KEY to generate diagonal shapes.

  4. Drag the mouse to draw the pixels.

Description Of My Technique

I separated the images into fourths and had shapes randomly

generate in each of the different four areas. I wanted to

create an effect with big and small shapes from half of the

canvas even though I split them to fourths. I added user

input like the using the keys to flip the image using

different shapes and sizes and to add more shapes. The

input is not restricted to the different areas, so the

shapes from the input go to any random spot. For mouse

input, if the user drags the mouse, two different shapes

will draw the pixels.
